main - main
mel_parser - лексер и парсер
mel_ast - узлы дерева
context - классы контекста и описания переменных и функция для него
tester - маленькие тестики (не очень качественные)
vartypes - перечисления для типа переменных, типа данных; классы для базовых типов и типа данных массива
typecaster - классы для приведения типов и функции для некоторые функции для логики приведения (есть подозрения, что можно упростить или даже совсем убрать)
bytecode_instructors - вспомогательный класс для генерации кода.
